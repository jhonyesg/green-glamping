📦 Desplegar el servicio de WhatsApp

Evolution API (carpeta whatsapp-evolution/):

cd whatsapp-evolution
cp .env.example .env   # define AUTHENTICATION_API_KEY
docker compose up -d
# Crear instancia y obtener QR:
curl -X POST http://localhost:8080/instance/create \
  -H "apikey: TU_KEY" -H "Content-Type: application/json" \
  -d '{"instanceName":"multibot","qrcode":true}'
