Proveedores de IA (LLM)

Tenant:

Cómo funciona la IA en Multibot: el bot responde primero con la base de conocimiento (regex, 0 costo, <100ms). El LLM se usa solo para lo que el regex no cubre: transcribir audios (STT), describir imágenes (vision), generar voz (TTS) y, a futuro, responder preguntas fuera de la KB. Si configuras varios proveedores, se usan en orden de prioridad con failover automático.
